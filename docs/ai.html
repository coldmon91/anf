AI 설정.

anf의 AI(요약·질문·이름 제안·자동 태그·정리·이미지 검색)는 기본 꺼짐입니다. 켜고 모델 하나만 연결하면 끝 — Claude가 가장 쉽고 강력합니다.

1. 설정 파일 열기

anf에서 ⌘, 또는 도구 메뉴 → AI 모델 연결…. 열리는 JSON 파일을 편집하고, anf으로 다시 전환하면 적용됩니다.

Claude (가장 추천)

1M 토큰 컨텍스트라 큰 문서·폴더도 통째로. 클라우드라 내용이 Anthropic으로 전송됩니다.

  1. console.anthropic.com → API Keys에서 Create Key. 키는 sk-ant-api03-… 형태입니다 (sk-ant-oat…는 OAuth 토큰이라 안 됩니다).
  2. 결제수단/크레딧이 등록돼 있어야 호출됩니다 (Billing).
  3. 설정 파일에 아래를 넣으세요:
"aiFeatures": true, "aiProvider": "auto", "aiApiKey": "sk-ant-api03-여기에-키"

모델은 비워두면 claude-opus-4-8이 기본입니다. "aiModel": "claude-sonnet-4-6"로 바꿀 수 있어요.

로컬 LLM (Ollama · LM Studio)

기기 밖으로 안 나갑니다. Apple Intelligence 없이도 동작.

  1. 로컬 서버를 켜세요 — Ollama는 자동(:11434), LM Studio는 Developer → Start Server(:1234).
  2. 설정 파일:
"aiFeatures": true, "aiProvider": "local", "aiEndpoint": "http://localhost:11434/v1", "aiModel": "llama3.2"

LM Studio는 http://localhost:1234/v1. aiModel은 로드한 모델 이름 (LM Studio는 비워도 됨). 추론 모델은 느릴 수 있어요.

Apple 온디바이스

번들·네트워크 0. macOS 26 + Apple Intelligence가 켜진 Mac이 필요합니다.

"aiFeatures": true, "aiProvider": "apple"

시스템 설정에서 Apple Intelligence를 켜고, Siri 언어가 맞아야 활성화됩니다.

쓰는 법

  • ⌘K → / 입력 → 현재 폴더에 바로 질문
  • 파일/폴더 우클릭 → AI 요약 · 질문하기 · 이름 제안 · 태그
  • 폴더 빈 공간 우클릭 / 도구 메뉴 → 폴더 요약·질문·정리

이미지 검색·OCR·스크린샷/종류별 정리는 Apple Intelligence 없이도 동작합니다. 문제가 있으면 GitHub Issues로 알려주세요.
